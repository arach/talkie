# Talkie Shared

Shared code, protocols, and data structures used across multiple platforms.

## Overview

This directory contains:
- Common data models and protocols
- Shared business logic
- Communication protocols between iOS/macOS and Backend
- Type definitions and interfaces

## Structure (Future)

```
Shared/
├── Models/          # Common data models
├── Protocols/       # Communication protocols
├── Types/           # Type definitions
└── Utils/           # Shared utilities
```

## Usage

Shared code can be:
- Swift Package for iOS/macOS
- TypeScript definitions for Backend
- Protocol buffers or JSON schemas for API contracts
