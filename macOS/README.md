# Talkie macOS

macOS companion application (planned).

## Overview

This will be a native macOS application that complements the iOS app, providing:
- Desktop notifications
- Menu bar integration
- Shared data sync via iCloud or Backend API
- Larger screen optimized UI

## Technology Stack (Planned)

- **Language**: Swift
- **UI**: SwiftUI (shared views with iOS where possible)
- **Target**: macOS 13.0+
- **Framework**: Native macOS app or Catalyst
