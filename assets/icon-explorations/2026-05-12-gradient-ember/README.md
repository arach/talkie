# gradient-ember

Landmark T filled with a top-to-bottom gradient — Hot Mic orange fading to deep cassette amber. Subtle warm ellipse glow behind the crossbar.

**What it evokes.** Captured warmth. The gradient feels like the T is heated from the top, glowing with stored voice energy. It's the filament inside a recording lamp: warm, alive, contained. The amber lower half grounds it in the cassette-era palette.

**Where it shines / where it fails.** Beautiful at 1024 and 512 — the gradient adds dimensionality without complexity. At 32px and below, the gradient collapses to a flat orange-brown and loses the ember distinction. Still reads as a T, but the emotional resonance fades. Best suited for iOS app icon and OG cards where the gradient can breathe.
