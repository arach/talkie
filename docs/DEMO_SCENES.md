# Talkie Demo Scenes

Concise demo scenarios for showcasing Talkie's core features.

---

## Scene 1: Quick Capture to Polished Note

**Flow:** Voice memo → Auto-transcription → Workflow polish

1. User taps record button on macOS menu bar
2. Speaks a rough idea: "I think we should add dark mode support, maybe start with the settings screen, then move to the main views"
3. Stops recording — transcription appears instantly
4. User runs "Polish" workflow — AI cleans up grammar, adds structure
5. Result: Clean, formatted note ready to share

**Highlights:** Speed of capture, seamless transcription, one-click refinement

---

## Scene 2: Hands-Free Keyboard Dictation

**Flow:** iOS keyboard → Background recording → Text insertion

1. User opens Messages app on iPhone
2. Taps Talkie keyboard, then mic button
3. Speaks message naturally while app switches to background
4. Returns to Messages — transcribed text auto-inserted
5. User sends without editing

**Highlights:** No context switch, works in any app, instant insertion

---

## Scene 3: Meeting Notes with Action Items

**Flow:** Long-form recording → Transcription → Extract workflow

1. User records a 5-minute meeting recap on Mac
2. Transcription completes, showing full text
3. User runs "Extract Action Items" workflow
4. AI returns bulleted list: owners, deadlines, next steps
5. User copies to task manager

**Highlights:** Handles longer recordings, AI extraction, practical output
